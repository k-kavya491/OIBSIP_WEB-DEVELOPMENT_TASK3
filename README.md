# Temperature Converter Web Application

## Objective

The objective of this project is to develop a simple and interactive **Temperature Converter** using **HTML, CSS, and JavaScript**. The application allows users to convert temperatures between **Celsius** and **Fahrenheit** (and optionally **Kelvin**) with accurate results. It also validates user input to ensure only numeric values are accepted, providing a user-friendly experience.

## Tools Used

* **HTML5** – To create the structure of the application.
* **CSS3** – To design a responsive and attractive user interface.
* **JavaScript** – To implement temperature conversion logic, input validation, event handling, and dynamic result display.
* **Visual Studio Code** – Code editor for development.
* **Google Chrome** – Browser for testing and debugging.

## Steps Performed

1. Designed the user interface with an input field, unit selection dropdown/radio buttons, a Convert button, and a result display area.
2. Applied CSS styling to create a responsive and visually appealing layout.
3. Implemented input validation to ensure only valid numeric values are accepted.
4. Developed JavaScript functions to convert temperatures between Celsius and Fahrenheit (with optional Kelvin conversion).
5. Used event listeners to trigger the conversion when the user clicks the **Convert** button.
6. Displayed the converted temperature with the appropriate unit.
7. Tested the application with different temperature values to ensure accurate conversions and proper validation.
